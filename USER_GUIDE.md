# 財金資訊系統使用說明

本文件是新版「多 Agent 台股智慧分析儀表板」的使用說明。新版主線為 FastAPI 後端加 Next.js 前端；根目錄 Streamlit 應用仍保留為 legacy MVP，不作為新版正式展示主線。

## 1. 線上使用

正式前端：

`https://multi-agent-tw-stock-dashboard-next.vercel.app`

使用流程：

1. 進入首頁。
2. 在搜尋框輸入台股代號，例如 2330、2317、2454。
3. 進入個股分析頁後，先看綜合判斷、信心分數與資料缺口。
4. 再依序查看摘要、技術面、基本面與估值、籌碼與風險、結論、辯論室與決策筆記。
5. 若畫面顯示資料降級或資料缺口，代表該次分析需要保守解讀。

## 2. 主要功能

- 股票搜尋與個股研究頁。
- 綜合判斷與信心分數。
- 資料缺口與資料品質提示。
- 技術面分析：收盤價、均線與 20 日報酬。
- 基本面與估值：EPS、PE、營收成長、歷史 PE 與規則式估值區間。
- 籌碼與風險：三大法人、融資融券、資料日期與反方觀點。
- 辯論室：整理支持觀點、反方風險與決策整合。
- 決策筆記：筆記只存在瀏覽器，不送到後端。

## 3. 資料來源

系統使用 TWSE / TPEx 官方資料、FinMind 與 yfinance。資料可能延遲、缺漏、限流或欄位格式改變；系統不宣稱資料完全即時。

## 4. 本機啟動

後端：進入 `api`，安裝依賴後以 uvicorn 啟動 FastAPI，預設位於 `http://127.0.0.1:8000`。

前端：進入 `next-dashboard`，安裝 Node 套件後啟動 Next.js，預設位於 `http://localhost:3000`。

Streamlit legacy：如需展示舊版 MVP，可在根目錄啟動 `app.py`。

## 5. API

- `GET /api/health`：檢查後端服務狀態。
- `POST /api/analyze`：取得指定股票的分析資料。

## 6. 安全規範

- 不提交任何私有環境設定、API key、token 或密碼。
- 不要求一般使用者在前端輸入 FinMind token。
- 不在畫面、log、README、報告或 GitHub issue 顯示真實 token。
- 資料不足時必須顯示資料缺口，不得硬補數據或估值。

## 7. 免責聲明

本系統分析結果僅供財金資訊系統課程、學術研究與投資參考，不構成任何買賣建議、獲利保證或專業投資顧問意見。投資人仍應自行評估風險，並自行承擔投資結果。
